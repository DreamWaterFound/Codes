# 使用Opencv进行的图片仿射变换的实验

----

#  一、功能简述

首先根据三对点之间的变化求出一个仿射矩阵，然后给定旋转中心和旋转角度生成一个旋转矩阵，使用cv::warpAffine()函数对图像进行仿射变换和旋转操作。比较简单。
